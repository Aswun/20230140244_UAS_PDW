# SIMPRAK: Sistem Informasi Manajemen Praktikum

## Penjelasan Singkat
SIMPRAK adalah sebuah aplikasi web yang dibuat untuk mengelola kegiatan praktikum, mulai dari pembagian materi, pengumpulan laporan, hingga penilaian tugas. Sistem ini dirancang untuk Mahasiswa dan Asisten, menyediakan fitur pendaftaran praktikum, pengumpulan tugas, melihat nilai, serta manajemen praktikum, modul, laporan, dan akun pengguna.
